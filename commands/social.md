# Social

Command | Description | Usage
--------|-------------|------
Award | Award points to a mentioned user | `award <member:user> <points:integer>`
Daily | Claim your daily points, with an option to set a reminder | `daily <reminder:string>`
Daily | Deduct points from a mentioned user | `deduct <member:user> <points:integer>`
Leaderboard | Show the server's leaderboard | `leaderboard`
Pay | Pay someone money | `pay <member:user> <points:integer>`
Score | Shows your score and level, with the option to see someone elses | `score [member:user]`
